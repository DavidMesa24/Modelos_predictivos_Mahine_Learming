4. Modelo de regresión Ridge¶

El objetivo de esta secci√≥n es mostrar c√≥mo construir un modelo de regresi√≥n con t√©cnicas de regularizaci√≥n, con ajuste de hiperpar√°metros con t√©cnicas de validaci√≥n. En este caso utilizaremos regularizaci√≥n norma L2, que en el contexto de regresi√≥n se conoce como Ridge. Se explicar√°, paso a paso, como:

  1. Preparar los datos para el modelado.
  2. Utilizar t√©cnicas de validaci√≥n para el ajuste de hiperpar√°metros.
  3. Construir un modelo con regresi√≥n Ridge.
  4. Evaluar el modelo.

Se utiliza el conjunto de datos relacionado con la venta de inmuebles.

Pero ¬øC√≥mo saber el mejor valor de alpha? Este hiperpar√°metro hay que ajustarlo a los datos y no podemos utilizar el conjunto test para realizar esta tarea. Debemos entonces recurrir a las t√©cnicas de validaci√≥n las cuales, en general, puedes implementarlas de dos maneras: utilizando un conjunto de validaci√≥n o empleando la validaci√≥n cruzada de k particiones (k-fold cross validation).

a) Con un conjunto de validaci√≥n. Si utilizas esta v√≠a el conjunto de entrenamiento hay que dividirlo en dos: uno para la construcci√≥n del modelo (ser√≠a el nuevo conjunto de entrenamiento) y el otro para la validaci√≥n (para determinar los valores de los hiperpar√°metros y complejidad).

Veamos c√≥mo hacerlo.
